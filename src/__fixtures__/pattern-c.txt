# Get started with Claude

Make your first API call to Claude and build a simple web search assistant

---

## Prerequisites

- An Anthropic [Console account](/)
- An [API key](/settings/keys)

## Call the API

```bash
export ANTHROPIC_API_KEY='your-api-key-here'
```

Make your first API call:

```bash
curl https://api.anthropic.com/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: $ANTHROPIC_API_KEY" \
  -d '{
    "model": "claude-sonnet-4-5",
    "max_tokens": 1000,
    "messages": [
      {
        "role": "user", 
        "content": "What should I search for?"
      }
    ]
  }'
```

## Endpoints

### Messages API

The Messages API is the primary way to interact with Claude.

```python
# Example code block with # characters
# This should not trigger a split
import anthropic

client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-sonnet-4-5",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello!"}]
)
```
